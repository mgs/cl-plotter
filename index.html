cl-plotter

Common Lisp Library For Interaction With Pen Plotters

View on GitHub Download .zip Download .tar.gz

About

cl-plotter is a library that provides a simple and direct route for working with Pen Plotters from Common Lisp.

Installation

The easiest way to install cl-plotter is via Quicklisp. From sbcl (ql:quickload "cl-plotter") will do the trick.
